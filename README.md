#COBOL examples
To compile the programs on a Linux computer, open the terminal in the main directory and type
	make
you will find the executables in the folder. To run one of the program just type
	./hello
###
hello.cbl
###
The classic "Hello World" program in cobol
###
sumnum.cbl
###
The program sumnum takes in input a positive integer n and compute and displays the sum of the first
n positive integers
###
salary_data.cbl
### 
The program salary_data create a file to be used as input to the salary program. The created file contains records
of employees (ID and SALARY).
###
salary.cbl
###
The program salary calculates the total of all salary amounts read from the file salary.dat
###
employee.cbl
###
The program employee reads data from the file employee.dat and displays them.
